# SEO Setup Instructions

## Google Search Console
1. Go to [Google Search Console](https://search.google.com/search-console/)
2. Add property: `https://sayeret-calculator.web.app`
3. Verify ownership using HTML file method or DNS
4. Submit sitemap: `https://sayeret-calculator.web.app/sitemap.xml`

## Google Analytics (Optional)
1. Create Google Analytics 4 property
2. Add tracking code to HTML files
3. Set up conversion goals for form submissions

## Bing Webmaster Tools
1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters/)
2. Add site: `https://sayeret-calculator.web.app`
3. Submit sitemap

## Local SEO (Israel)
1. Create Google My Business listing (if applicable)
2. Add Hebrew language targeting in Search Console
3. Set geographic targeting to Israel

## Social Media
1. Create Facebook page for the business
2. Set up WhatsApp Business (if applicable)
3. Add social media links to website

## Content Optimization
1. Add more Hebrew keywords to meta descriptions
2. Create blog content about mortgage tips (optional)
3. Add FAQ section with common mortgage questions
4. Include customer testimonials/reviews
